### Mock服务器端返回

使用Ruby on Rails框架开发一个服务器端返回JSON(P)返回值。一般适用于前后端分离的项目。

你可以Mock:

* JSON(P)数据

* 图片返回

* 不同状态码

* ...

### 工作流程

      |----------------|                      |-----------------|
      |                |     Inject mock.js   |                 |
      |  Local Dev Env | -------------------> | Server Side API | 
      |                |                      |                 |
      |----------------|                      |-----------------|
            /||\                                      ||
             ||                                       ||
             ||                                      \||/
             ||                               |-----------------|
             ||       JSON(P)                 |                 |  
             ||-------------------------------|Ajax Mock Server |   { Configurable, Flexiable }
                                              |                 |
                                              |-----------------|
